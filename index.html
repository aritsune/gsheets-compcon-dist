LANCER Google Sheet to comp/con converter

This tool converts LANCER characters from the LANCER discord's Google Sheets sheet to beeftime's comp/con tool.

Instructions:

  1. On your Google sheet, go to File > Download as > Microsoft Excel (.xlsx)
  2. Upload it into this tool
  3. Copy the results to clipboard
  4. In comp/con, add a new pilot, and select Import from clipboard
